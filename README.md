# vapper (WIP)

[![lerna](https://img.shields.io/badge/maintained%20with-lerna-cc00ff.svg)](https://lerna.js.org/)

## Setup

```sh
git clone https://github.com/vapperjs/vapper.git
cd vapper
yarn install
```

## Run demo

### 1、link

```sh
cd packages/cli
yarn link
```

### 2、run

```sh
cd examples/vue-cli3
yarn install

# dev
vapper dev

# build
vapper build

# start the production server
vapper start
```
